# Шахматы в консоли (C#)

Простая реализация шахматной игры в текстовом формате на языке C#. Игра позволяет двум игрокам поочередно совершать ходы, проверяет корректность ходов, шах, мат и пат.

## Особенности

- Игровое поле 8x8 с начальной расстановкой фигур.
- Визуализация доски в консоли с использованием юникодных символов шахматных фигур.
- Ввод ходов в формате e2e4 (например, с поля e2 на поле e4).
- Проверка корректности хода с учетом правил шахмат (включая проверку на шах).
- Обнаружение шаха, мата и пата.
- Правильная смена хода между белыми и черными.

## Управление

- Вводите ходы в формате ab cd, где ab — начальная позиция, cd — конечная позиция (например, e2e4).
- Позиции обозначаются по шахматной нотации: a-h по горизонтали, 1-8 по вертикали (1 — нижний ряд для белых).

## Структура проекта

- Program.cs — основной файл с логикой игры.
- Piece — перечисление (enum) всех возможных фигур на доске.

## Логика игры

- Игра начинается с расстановки фигур.
- Игроки поочередно вводят ходы.
- Проверяется, может ли фигура совершить ход по правилам.
- Проверяется, не окажется ли король под шахом после хода.
- При шахе/мате/пате игра завершается.
